# Player Ratings Import Script

This script downloads player ratings an external api and convert them to a format that can be used in game.

## Features
- Reads ratings from a specified API.
- Downloads team logos
- Download player avatars

---

## Requirements
- Python 3.8+


## Usage

### Command-line
```bash
python ratings.py
```
